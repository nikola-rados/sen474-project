# Components
## Data
### `heroes.json`
Hero names and IDs.

### `training_data.json`
DotA 2 match data used for training.

### `testing_data.json`
DotA 2 match data used for testing.

## Code
### `collect.py`
Responsible for collecting DotA 2 match data from Steam API.

### `process.py`
Responsible for processing the data collected.
